## 4. CINEMATIC LANGUAGE — Shot Types & Storytelling

> Pendekatan sinematografi: setiap slide = 1 frame dari film.
> Gunakan visual storytelling, bukan hanya "gambar yang cocok dengan teks".

### 4.1 Shot Types (Wajib dipilih per slide)

| Shot | Framing | Use Case | Prompt Keyword |
|:---|:---|:---|:---|
| **Establishing** | Full landscape, karakter kecil | Buka section baru, perkenalkan lokasi | `wide establishing shot` |
| **Wide** | Full body, environment visible | Perjalanan, caravan, battle overview | `wide shot` |
| **Medium** | Waist up, some environment | Dialog, interaksi 2-3 orang | `medium shot` |
| **Close-up** | Face/chest, minimal bg | Emosi kuat, momen penting | `close-up shot` |
| **Over-shoulder** | From behind 1 karakter, melihat ke karakter lain | Percakapan, konfrontasi | `over-the-shoulder shot` |
| **Bird's eye** | Top-down view | Skala pertempuran, kota, geografi | `bird's eye view` |
| **Low angle** | Dari bawah melihat ke atas | Power, kekuatan, keagungan | `low angle shot` |
| **Detail/Insert** | Objek close-up (tangan, surat, pedang) | Simbol, objek kunci cerita | `detail shot of [object]` |

### 4.2 Sequence Pattern — Rhythm per Section

Setiap section (2-3 slides) harus mengikuti **rhythm sinematik**:

```
Section pattern A (standard):
  Slide 1: ESTABLISHING/WIDE → set lokasi + waktu
  Slide 2: MEDIUM → aksi/dialog utama
  Slide 3: CLOSE-UP → reaksi emosional / punchline

Section pattern B (action/tension):
  Slide 1: WIDE → konteks
  Slide 2: MEDIUM → konfrontasi
  Slide 3: LOW ANGLE atau DETAIL → klimaks

Section pattern C (reflective/lesson):
  Slide 1: WIDE → landscape contemplative
  Slide 2: MEDIUM → karakter reflecting
  (or single ESTABLISHING shot for final slide)
```

**Rule**: JANGAN pernah 3 wide shots berturut-turut. Alternasi wajib.

### 4.3 Genre-Specific Visual Language

Project ini punya event beragam (perjalanan, perang, politik, spiritual). Setiap genre punya bahasa visual berbeda:

#### 🐪 Journey/Travel Events (e03, e09, e15)
```
Palette: warm golds, sandy browns, blue sky
Shots: dominan wide + establishing (skala perjalanan)
Mood: adventure, vastness, progress
Key frame: caravan silhouette against sunrise/sunset
Lighting: natural outdoor, golden hour dominant
```

#### ⚔️ Battle/Conflict Events (e19, e20, e21, e47, e110, e111)
```
Palette: dramatic contrast — dark clouds + bright armor/banners
Shots: wide → medium → low angle (escalation)
Mood: tension, bravery, sacrifice
Key frame: two forces facing each other
Lighting: dramatic — storm clouds, dust, shafts of light
Movement: diagonal composition (cavalry charging, arrows flying)
⚠️ Children-friendly: show BRAVERY not VIOLENCE. 
   No blood, no graphic injury. Focus on courage, formation, strategy.
```

#### 🕌 Spiritual/Miracle Events (e06, e12, e34)
```
Palette: deep blues + divine gold + pure white
Shots: low angle (looking up at heavens), wide (landscape + sky)
Mood: awe, wonder, divine presence
Key frame: light descending from above
Lighting: supernatural — golden glow from above, starlight
Movement: vertical (ascending, descending)
```

#### 🏛️ Political/Governance Events (e16, e32, e35, e42)
```
Palette: earth tones + authority colors (deep blue, green)
Shots: medium dominant (councils, meetings, declarations)
Mood: gravity, wisdom, authority
Key frame: leader addressing crowd, or document/treaty scene
Lighting: formal interior — window light, lamp light
Composition: symmetrical (authority) or triangular (negotiation)
```

#### 💔 Hardship/Persecution Events (e10, e108, e44)
```
Palette: muted/desaturated — grey, dark blue, faint warm
Shots: close-up dominant (suffering), medium (group endurance)
Mood: sorrow BUT resilience, never despair
Key frame: single figure enduring, with a light (hope) somewhere
Lighting: low-key — fire in darkness, single lamp, moonlight
⚠️ Children-friendly: show STRENGTH in suffering, not graphic pain.
   Tears OK, blood NO. Focus on "Ahad" — faith unbroken.
```

#### 🎓 Civilization/Knowledge Events (e46, e47-andalus, e112)
```
Palette: vibrant — red, green, blue, gold (prosperity)
Shots: wide establishing (cities, libraries), medium (scholars)
Mood: wonder, achievement, beauty
Key frame: grand architecture, bustling markets, books/scrolls
Lighting: bright, warm, abundant — daytime dominates
```

### 4.4 Scene Differentiation (Anti-Redundancy)

> Problem: 2+ slides di lokasi sama → gambar terlihat identik.

**Minimum 2 dari 3 differentiator wajib berubah antar slide:**

1. **Shot type** (lihat 4.1)
2. **Focus subject** (siapa/apa di foreground)
3. **Lighting direction / intensity**

**Anti-Pattern Checklist:**
- [ ] Tidak ada 2 slide bersebelahan yang sama-sama wide shot
- [ ] Tidak ada 2 slide di lokasi sama dengan angle identik
- [ ] Karakter foreground BERBEDA dari slide sebelumnya (jika di lokasi sama)
- [ ] Waktu/lighting bergeser logis (tidak siang-malam-siang tanpa transisi)

### 4.5 Storyboard Planning (WAJIB sebelum generate)

Sebelum menulis prompt, buat **storyboard table** per event:

```
| # | Section | Shot | Focus | Location | Time | Mood | Differentiator vs prev |
|---|---------|------|-------|----------|------|------|------------------------|
| 1 | Intro   | EST  | landscape | Mecca | sunrise | hope | — |
| 2 | Intro   | MED  | Abu Thalib | courtyard | morning | love | angle + focus |
| 3 | Intro   | CU   | golden glow | courtyard | morning | joy | angle + focus |
| ...
```

Storyboard ini jadi kontrak — prompt harus match, QA harus verify.

## 6. PROMPT ASSEMBLY TEMPLATE

> Gunakan template ini untuk setiap prompt. Copy-paste dari sections di atas.
> Order matters — Gemini membaca dari atas ke bawah, prioritas di awal.

```
Create an image: A warm watercolor storybook illustration for children. 
Image ratio: exactly 16:9 landscape (wide). Resolution: 1792x1024.

SHOT: [establishing/wide/medium/close-up/over-shoulder/low-angle/bird's-eye/detail]
GENRE PALETTE: [journey/battle/spiritual/political/hardship/civilization — see Section 4.3]
TIME: [sunrise/morning/afternoon/golden hour/dusk/night — see Section 3]

LOCATION: [paste dari Section 2]
SCENE: [specific action/moment — 2-3 sentences]

[CHARACTER 1]: [paste dari Section 1]
[CHARACTER 2]: [paste dari Section 1, if applicable]
[GOLDEN GLOW]: [paste dari Section 1.1, with correct height for age]

[SUPPORTING]: [camels, trees, props — paste dari Section 5]

Colors: [from genre + time-of-day palette]
Mood: [1-2 words]

COMPOSITION: [shot type] framing. Place all important subjects within 
the CENTER 70% of the image. Leave outer edges clear of important details.
CRITICAL: Do NOT include ANY text, words, letters, labels, captions, 
titles, signatures, or writing of any kind in the image.
```

### Prompt Length Rule
- **Max ~600 characters** — lebih panjang sering gagal di Gemini (no Send button)
- Kalau terlalu panjang → prioritaskan SHOT + SCENE + CHARACTER, potong supporting details
- Character description bisa disingkat ke 1 line jika space terbatas
